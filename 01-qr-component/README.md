# QR Component
Welcome to my first Frontend Mentor challenge! In this project, I have designed a QR code component using pure HTML and CSS. Throughout this task, I gained valuable insights into centering elements using flexbox, importing fonts from Google Fonts, and applying various CSS styles to create an engaging and visually appealing QR component.

**Technologies Used**
- HTML
- CSS

## Key Takeaways

- Utilized flexbox for efficient centering of elements.
- Incorporated Google Fonts to enhance the typography.
- Demonstrated proficiency in applying diverse CSS styles.
